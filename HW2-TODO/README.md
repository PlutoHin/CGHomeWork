项目名称：
**WebGL三维模型场景与交互实验**

------

**一、项目简介**

本项目基于 WebGL 实现了多个三维模型的绘制与交互控制，包括基本的草帽模型、立方体、球体以及附加实现的太阳系模型。通过对顶点数据、PVM矩阵变换及交互逻辑的编写，项目完成了从建模、渲染到交互控制的完整流程，展示了三维图形渲染的核心原理。

------

**二、主要功能**

1. **基础模型绘制（TODO1）**
   - 实现了网格模型“草帽”的建模与渲染，顶点与颜色由程序生成并传入 GPU。
2. **交互控制功能（TODO2）**
   - Z / C：控制模型整体缩放
   - W / A / S / D：控制相机绕 X、Y 轴旋转
   - P：在透视投影与正投影之间切换
   - M / N：调整视角大小
   - 鼠标拖动：控制相机视角方向
3. **渲染控制功能（TODO3）**
   - R / T：开启或关闭后向面剔除
   - B / V：开启或关闭深度缓存消隐
4. **附加功能**
   - 实现了WASD控制物体的运动，鼠标控制相机运动；
   - 实现了太阳系模型，包括太阳与地球两个球体，地球绕太阳旋转；
   - 下拉框选择不同模型（立方体、球体、草帽、太阳系）进行切换；
   - 模型外观更精致，颜色区分清晰，球体采用细分四面体算法生成。

------

**三、运行方式**

1. 将项目文件放入本地服务器环境。
2. 打开浏览器运行 `3d-wandering.html`。
3. 页面加载后，通过下拉菜单选择模型；
4. 使用键盘与鼠标进行交互控制，体验不同模型与投影模式。

------

**四、文件结构说明**

```
│
├── 3d-wandering.html          // 主页面，包含下拉框与画布
├── 3d-wandering.js            // 主逻辑文件，模型绘制与交互控制
├── models-data.js             // 存放各模型顶点和颜色数据
├── Common/
│   ├── webgl-utils.js         // WebGL 工具库
│   ├── initShaders.js         // 着色器初始化
│   └── MVnew.js               // 矩阵与向量运算库
```

------

**五、操作说明**

1. **模型切换**：通过网页上方的下拉框选择模型（Cube、Sphere、Hat、Solar System）。
2. **场景控制**：
   - 移动相机：WASD
   - 缩放模型：Z / C
   - 投影切换：P
   - 视角调节：M / N
   - 鼠标拖拽：旋转视角
3. **渲染选项**：
   - R/T：开启或关闭后向面剔除
   - B/V：开启或关闭深度缓存消隐
